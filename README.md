# GOAP
<b>Roșu Mihai Cosmin</b>

Project made for the Algorithms for Video Games Logic course of the Graphics, Multimedia and Virtual Reality master program.

## Details

Small 2D top-down game implementing the Goal Oriented Action Planning system for the enemy AI.

In the game the player has to defeat a single enemy by moving around the map and picking up collectibles such as <b>heals</b>, <b>speed-ups</b> and <b>swords</b>. The enemy can pick up the same collectibles and the GOAP system determines its next course of action.

A more detailed documentation along with results can be found at: [ProiectALJV_RosuMihaiCosmin_Rezultate.pdf](https://github.com/mehigh8/GOAP/blob/main/ProiectALJV_RosuMihaiCosmin_Rezultate.pdf)
